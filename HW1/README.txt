How to run Homework#1

1. Open up either a command line window or a Powershell window
2. Navigate to the folder containing your Python Script 
3. Then type "Python "script_name" "File that contains biological sequence" "File where you want the output"
**When running the code to find out what the biological sequence is you output file part in step 3 can be skipped.
**When running script to search for substrings, enter
	python subsearch.py [substring] [target file]
